## [1.0.0] - 2026-03-24

### 新增
- 底部导航：首页、日历、回顾、设置。
- 心事记录：新建 / 编辑 / 详情 / 删除；心情选择；标签；本地 SQLite 存储。
- 首页搜索（正文与标签）。
- 日历月视图与按日列表；本周回顾与心情分布。
- 应用锁：数字 PIN（安全存储摘要）、可选生物识别；从后台回到前台可再次锁定。
- 主题：跟随系统 / 浅色 / 深色。
- Web 端通过 `sqflite_common_ffi_web` 初始化数据库，避免白屏。

### 说明
- 无云同步、无 AI（MVP 范围）。

发布新版本时可在 GitHub 上打 `v1.0.0` 等标签，并把「未发布」下的条目移到带日期的版本标题下。
